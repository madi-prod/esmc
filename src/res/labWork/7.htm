РЕШЕНИЕ ОБЫКНОВЕННЫХ ДИФФЕРЕНЦИАЛЬНЫХ УРАВНЕНИЙ.

 

Задание1. Применяя метод Эйлера, численно решить следующие дифференциальные уравнения с данными начальными условиями на отрезке  с шагами  и  при указанных значениях параметров.

Задание2. Методом Рунге-Кутта найти приближённое решение дифференциального уравнения при заданных начальных условиях с указанным шагом  в первых шести точках. Оценить погрешность.

Задание3. Методом Пикара найти приближённое решение дифференциального уравнения , удовлетворяющего начальному условию  либо из задания 1, либо из задания 2 с точностью . Провести сравнительный анализ.

Задание4. По таблицам, полученным в заданиях 1-3, построить графики интегральных кривых.
